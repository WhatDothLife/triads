#+TITLE: Triads

* About
A program for generating core triads and checking polymorphisms.

* Installation
~git clone https://github.com/hlissner/doom-emacs.git~
~cargo build~

* Usage
Use ~--help~

Example usage: ~./triads -p commutative -l 8~
